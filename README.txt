(este programa não se encontra funcionando desde a ultima atualização do site ligamagic.com)

Estes programas tem como função acelerar o processo de recuperação de dados de cartas de sites ligados ao 'LigaMagic'.



* PARA O USO MAIS FÁCIL DO PROGRAMA DE EXTRAÇÃO DE DADOS DE CARDS DE UMA LOJA, entre no link https://repl.it/@isabelacostasou/Cardretriever, coloque no arquivo 'mtg_cards.txt' as cartas desejadas e clique em 'RUN', seu resultado estará no arquivo 'card_prices.txt' *

* PARA O USO MAIS FÁCIL DO PROGRAMA DE COMPARACAO DE PRECOS ENTRE LOJAS, entre no link https://repl.it/@isabelacostasou/Storecomparation, coloque no arquivo 'mtg_cards.txt' as cartas desejadas e clique em 'RUN', seu resultado estará no arquivo 'store_comparation.txt' *


Sobre os programas:


ligamagic_card_retriever (extrai preços de cartas e suas quantidades disponíveis de uma loja):

    Para que este funcione, basta passar criar um arquivo 'mtg_cards.txt' com todas as cartas desejadas, separadas por quebras de linha e informar via código o link principal da loja da qual deseja obter dados (preco, tipo e quantidade disponível). 

    Com isso pronto, basta rodar o arquivo 'ligamagic_price_retriever.py', que criará um arquivo 'card_prices.txt' com todas as informações das cartas informadas.
    

ligamagic_card_comparation (faz a comparação de preços de cartas em duas lojas diferentes, retornando a melhor opcao de compra):

Para que este funcione, basta passar criar um arquivo 'mtg_cards.txt' com todas as cartas desejadas, separadas por quebras de linha e informar via código o link principal da loja da qual deseja obter dados (preco, tipo e quantidade disponível). 

Com isso pronto, basta rodar o arquivo 'ligamagic_card_comparation.py', que criará um arquivo 'store_comparation.txt' com a comparação realizada entre as lojas.

PS.: Se você não tem conhecimento e/ou um ambiente virtual para rodar programas Python, basta fazer upload do programa e do arquivo .txt com as cartas desejadas no site 'repl.it/languages/python3' e clicar 'run', o código será executado e você terá seu arquivo com as informações das cartas da mesma forma
